# موقع إبراهيم - Laravel Portfolio

موقع شخصي للمطور إبراهيم يعرض أعماله ومهاراته التقنية.

## الميزات

- **تصميم متجاوب**: يعمل على جميع الأجهزة
- **نظام إدارة محتوى**: لوحة تحكم لإدارة المحتوى
- **مدونة تقنية**: مقالات وأخبار تقنية
- **نموذج اتصال**: للتواصل مع العملاء
- **واجهة عربية**: دعم كامل للغة العربية

## متطلبات النظام

- PHP 8.1 أو أحدث
- Composer
- Node.js و NPM
- MySQL أو SQLite
- Laravel 11

## التثبيت

1. **استنساخ المشروع**:
```bash
git clone <repository-url>
cd laravel-app
```

2. **تثبيت التبعيات**:
```bash
composer install
npm install
```

3. **إعداد البيئة**:
```bash
cp .env.example .env
php artisan key:generate
```

4. **إعداد قاعدة البيانات**:
قم بتحديث إعدادات قاعدة البيانات في ملف `.env`

5. **تشغيل الترحيلات**:
```bash
php artisan migrate
php artisan db:seed
```

6. **بناء الأصول**:
```bash
npm run build
```

7. **تشغيل الخادم**:
```bash
php artisan serve
```

- **Laravel 11**: إطار العمل الخلفي
